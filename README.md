🧠 AI Resume Analyzer

A smart web application that analyzes resumes and provides skill-based suggestions, missing skills, and career guidance.

---

🚀 Features

- 📄 Upload Resume (PDF / DOCX / Text)
- 🔍 Extracts content from resume
- 📊 Calculates Resume Score
- ✅ Detects existing skills
- ❌ Shows missing skills
- 💡 Provides suggestions to improve resume
- 🎯 Role-based analysis (Data Analyst, Web Developer, etc.)

---

🛠️ Tech Stack

- Frontend: HTML, CSS, JavaScript
- File Processing: PDF.js, Mammoth.js
- Deployment: GitHub Pages

▶️ How to Use

1. Upload your resume (PDF/DOCX)
2. Select job role
3. Click Analyze Resume
4. View score, skills, and suggestions

---

📈 Sample Output

- Resume Score: 50/100
- Matched Skills: Python, Excel
- Missing Skills: SQL, Power BI
- Suggestions: Improve missing skills and add projects
